What is latitude?

Latitude is a geographic coordinate that specifies the north or south position of a point on the earth's surface. It is the measurement of distance from the equator. Latitude contributes to the weather.

The purpose of this project is to analyze what the weather is like based on latitudes using the software tools that we learned in data bootcamp: Python, Matplotlib, Pandas, API, HTML, Bootstrap and CSS.

Using Citypy Python library, we gathered a dataset of over 500 cities to be used for this analysis. Then, using OpenWeatherMap API, we gathered the data of the selected cities to plot the weather data using Matplotlib.

And finally, the presentation of results and observations is written in HTML with the use of Bootstrap and CSS.
